Semantic tags

HTML5 is based around semantic tags rather than just calling everything 'div'. According to the Manchester codes site:

Semantic HTML tags do little (and often nothing) to modify their contents appearance. However, they make our code easier to read for other developers, search engines and screen readers. Manchester Codes

An article contains something that could stand alone or be reused, whereas an aside contains content not directly related to the main content, e.g. advertising.
